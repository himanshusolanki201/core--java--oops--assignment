## Overview

This is a Java console-based application built for a training institute to manage student information, employee details, product data, and billing processes. The project strictly adheres to Object-Oriented Programming (OOP) principles.

## Concepts Used

- **Classes & Objects:** For structuring data (Student, Employee, Product).
- **Constructors & `this` Keyword:** For direct object initialization.
- **Encapsulation:** Implemented in the `Product` class using `private` variables and Getter/Setter methods.
- **Inheritance:** Demonstrated using the `CollegeStudent` class extending the `Student` class.
- **Polymorphism (Method Overloading):** Handled in the `ProductBilling` class to process different numbers of products.
- **`static` Keyword:** Used in the `Employee` class for a shared company name, and in the `Student` class to track the total student count.
- **Arrays & Loops:** Used to store and iterate through student marks.
- **String Manipulation:** Used to format and analyze student names.
